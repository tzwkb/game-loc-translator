# 项目专属规则占位文件

此文件为占位模板。实际项目中，复制此文件并命名为：
- `translate_project_{PROJECT_ID}.md` — 用于全新翻译模式
- `optimize_project_{PROJECT_ID}.md` — 用于初译优化模式

## 项目专属规则应包含

1. **世界观设定**：时代背景、核心势力、关键事件
2. **角色规则**：主要角色名、称谓、口癖
3. **术语覆盖**：项目特有术语的译法说明
4. **风格细则**：比风格指南更具体的本项目要求
5. **禁用词**：本项目绝对不能出现的表述
6. **格式特殊要求**：如标签处理、换行规则

## 示例结构

```markdown
## 世界观
本项目为古风仙侠，设定在"玄黄大陆"，分九大门派...

## 核心角色
- 主角：李青云 — 昆仑派弟子，性格沉稳
- 反派：血魔老祖 — 血海教教主，自称"老夫"

## 术语说明
- 玄黄气 → 不可译为"真气/内力"，固定为"玄黄气"
- 渡劫 → 修仙者突破境界时的天劫，不可简写

## 风格要求
- 所有技能描述必须四字对仗
-  dialogue中禁止使用"你"，改用"阁下/足下"

## 禁用词
- 能量（过于科幻）
- 升级（过于游戏化，改用"突破"）
```
