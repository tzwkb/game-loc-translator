# Quality Acceptance Checklist

## Terminology Check (Must Be 100% After Glossary Enforcement)

- [ ] All glossary terms appear in correct form
- [ ] No glossary term has been replaced with synonym
- [ ] No glossary term has been abbreviated without authorization
- [ ] Variant forms (plural, possessive, lowercase) handled consistently

## Style Check

- [ ] First-paragraph tone maintained throughout
- [ ] Dialogue matches character voice established early
- [ ] Skill descriptions follow genre conventions
- [ ] Narrative maintains consistent register

## Context Check

- [ ] No contradictions with knowledge base lore
- [ ] Character names and titles consistent with setting
- [ ] Cultural references appropriate for target audience
- [ ] No anachronisms in historical/fantasy settings

## Format Check

- [ ] Game tags ({b}, {/b}, {nl}, etc.) preserved exactly
- [ ] No truncation or cut-off sentences
- [ ] Newline sequences (\n) preserved as literal
- [ ] No unexpected markdown or code fences in output

## Mode B Specific (MTPE)

- [ ] At least 85% of reasonable drafts preserved unchanged
- [ ] No over-optimization (polishing good sentences)
- [ ] Change annotations present and accurate
- [ ] Diff output matches actual changes

## Scoring

| Dimension | Weight | Threshold |
|-----------|--------|-----------|
| Terminology | 40% | 100% |
| Style | 25% | >= 90% |
| Context | 20% | >= 90% |
| Format | 15% | 100% |

**Overall Pass:** >= 90%
**Rewrite:** 70-89%
**Manual:** < 70%
